# Инструкция по использованию

## Графический интерфейс
1. Запустите приложение: python main.py
2. Выберите режим работы:
   - "Примеры" - для решения готовых уравнений из базы
   - "Своё уравнение" - для ввода своего уравнения
3. Для режима "Примеры":
   - Выберите уравнение из выпадающего списка
   - Нажмите "Построить график" для отображения графика
   - Нажмите "Сохранить график" для сохранения в папку images/
   - Нажмите "Сохранить результат" для сохранения аналитического решения
4. Для режима "Своё уравнение":
   - Введите уравнение, например: a*x + b = 0
   - Нажмите "Решить"
   - Нажмите "Построить график" для отображения графика
   - Нажмите "Сохранить график" для сохранения в папку images/
   - Нажмите "Сохранить результат" для сохранения аналитического решения

## Командная строка
# Для своего уравнения
python main.py --eq "a*x + b = 0"

# Для примера из базы
python main.py --example "linear/1"
python main.py --example "quadratic/2"

## Поддерживаемые уравнения
- Линейные: a*x + b = 0
- Квадратные: a*x^2 + b*x + c = 0

Приложение автоматически определяет тип уравнения и применяет соответствующий метод решения.
